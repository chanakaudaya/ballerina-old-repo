
Ballerina is a new programming language for integration built on a sequence diagram metaphor.
This command mainly used to execute the main function of a ballerina program.

Usage:
    ballerina.bat ballerina-file [args...]
    ballerina.bat command

Examples:
    ballerina.bat hello-world.bal WSO2
    ballerina.bat hello-world.bal "WSO2 Inc"  (If the argument value contains a space, use double quote to enclose it)
    ballerina.bat multiplier.bal 5 10
    ballerina.bat version
    ballerina.bat help

Available Commands:
    version     Print Ballerina version.
    help        Print this help message.
